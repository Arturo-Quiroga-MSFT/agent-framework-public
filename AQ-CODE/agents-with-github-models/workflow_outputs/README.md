# Workflow Outputs

This directory stores the formatted outputs from workflow executions:

- `sequential_github_*.txt` - Outputs from Example 07 (sequential workflow)
- `parallel_github_*.txt` - Outputs from Example 08 (parallel workflow)

Each file is timestamped with format: `YYYYMMDD_HHMMSS`

These outputs are saved automatically when you run the workflows through DevUI.
